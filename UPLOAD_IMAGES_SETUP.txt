========================================
SETUP UPLOAD GAMBAR PRODUK
========================================

Sistem upload gambar sudah diubah dari URL menjadi UPLOAD FILE dari galeri.
Setiap produk sekarang punya:
- 1 THUMBNAIL (gambar utama untuk catalog)
- 4 GALLERY IMAGES (preview di detail produk)

========================================

========================================
STRUKTUR FILE:
========================================

Folder uploads:
D:\test\uploads\                    (untuk development)
C:\laragon\www\test\uploads\        (untuk Laragon)

Gambar tersimpan dengan nama unik:
- Format: {uniqid}_{timestamp}.{ext}
- Contoh: 6567a8f3_1732265843.jpg

========================================
CATATAN PENTING:
========================================

1. Produk LAMA masih pakai URL gambar
   - Bisa tetap ditampilkan
   - Edit produk lama untuk upload gambar baru

2. File type yang diizinkan:
   - JPG, JPEG, PNG, GIF, WEBP

3. Ukuran file:
   - Maksimal tergantung php.ini (default 2MB)
   - Bisa diubah di php.ini: upload_max_filesize

4. Folder uploads harus writable:
   - Windows: sudah otomatis
   - Linux: chmod 755 atau 775

========================================
TROUBLESHOOTING:
========================================

Q: Error "Failed to upload"?
A: - Cek folder uploads/ ada dan bisa ditulis
   - Cek php.ini: upload_max_filesize & post_max_size
   - Restart Laragon

Q: Gambar tidak muncul di catalog?
A: - Cek path gambar di database (harus: uploads/filename.jpg)
   - Cek file fisik ada di folder uploads/
   - Hard refresh browser (Ctrl+F5)

Q: Edit produk lama belum bisa upload?
A: - Fitur edit akan diupdate nanti
   - Saat ini bisa hapus dan buat ulang produk
