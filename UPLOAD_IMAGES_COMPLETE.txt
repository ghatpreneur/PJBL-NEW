========================================
TROUBLESHOOTING:
========================================

Q: Error "Failed to upload"?
A: Cek folder uploads/ ada dan writable

Q: Gambar tidak muncul?
A: - Cek path di database (harus: uploads/filename.jpg)
   - Cek file ada di folder uploads/
   - Hard refresh (Ctrl+F5)

Q: Preview tidak muncul saat edit?
A: Path gambar harus relative ke edit_product.php
   (gunakan ../ jika perlu)

Q: File terlalu besar?
A: Edit php.ini:
   upload_max_filesize = 10M
   post_max_size = 10M
